#  Indian Economy Health Analyzer

The Indian Economy Health Analyzer is a data-driven web application built using Python and Streamlit that helps visualize and evaluate the overall economic condition of India. The project focuses on key economic indicators such as GDP growth rate, inflation rate, unemployment rate, and Sensex performance to provide a simplified understanding of economic trends over time.

The application processes and cleans the dataset using Pandas and NumPy, and presents insights through interactive visualizations created with Plotly. It also computes a combined economic health score based on multiple indicators, allowing users to quickly assess whether the economy is performing well or facing challenges.

The interface is designed to be user-friendly and visually appealing, enabling users to explore trends, compare data, and gain meaningful insights without needing technical knowledge. This project demonstrates practical implementation of data analysis, visualization, and web deployment using modern Python tools.

### 🔧 Tech Stack

* Python
* Pandas & NumPy
* Streamlit
* Plotly

### 🚀 Features

* Interactive dashboard for economic indicators
* Data cleaning and preprocessing
* Visualization of trends using graphs
* Economic health score calculation
* Simple and responsive UI
